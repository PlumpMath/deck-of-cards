
# Deck of Cards

[![Build Status](https://secure.travis-ci.org/scotthaleen/deck-of-cards.png?branch=master)](http://travis-ci.org/scotthaleen/deck-of-cards)

Java 8 Deck of Cards

### Test

```
gradle test
```

### Build

```
gradle build
```

### Run 

```
java -cp  .:build/libs/deck-of-cards.jar deck.Main
```

**with gradle**

```
gradle runMain
```
